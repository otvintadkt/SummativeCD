**Название: Картографическая смекалка**

**Суть игры:**
Надо повернуть руки в сторону, которая будет указана на экране, представив, 
что перед вами карта, где сверху север.
Это развивает как ориентирование на карте, так и на местности.

**Правила:**
Надо в ту сторону света, которая показана на экране, показать две или одну руку (север и юг - обе руки веерх или вниз
соответственно, запад - левая рука влево, восток - правая рука вправо). Когда требуется показать восток нельзя
показывать запад левой рукой и наоборот!
Начать и закончить раунд можно на пробел. Выйти из игры можно на клавишу Q. Есть два режима: бесконечный режим и режим

**Установка:**
Поддерживаемые ОС: Windows 10/11
Требуется установленный python >= 3.10, pip==25.3
1. Перейти по ссылке https://github.com/otvintadkt/SummativeCD/archive/refs/heads/main.zip, скачать файл.
2. Распаковать архив в нужную вам папку
3. Открыть распакаванную папку (может открыться автоматически), в ней открыть ещё одну.
Должна открыться папка содержащая main.py, config.py и др.
4. Скопировать путь к этой папке (например, D:\game\SummativeCD-main\SummativeCD-main)
5. Нажать комбинацию клавиш win+R, прописать cmd в появившемся окне. Откроется терминал.
6. Если папка с игрой находится на другом диске, прописать в терминале его название (например, D:)
7. В терминале написать `cd <путь_к_папке_с_игрой>`
8. Пропишите команду <путь_до_python.exe> -m venv venv. Чаще всего python расположен здесь:
C:\Users\<имя_пользователя>\AppData\Local\Programs\Python\Python<версия>.
9. Подождите, пока установится папка venv
10. Активировать окружение: `venv\Scripts\activate.bat`
11. Установить зависимости: `venv\Scripts\python.exe -m pip install -r requirements.txt`
12. Дождитесь установки
13. Установить OpenCV: `pip install opencv-python==4.12.0.88 --no-deps`
14. После установки всех библиотек для запуска игры пропишите python main.py
